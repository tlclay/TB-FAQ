Set Commands

Q: What are set commands?

A: This category includes ones which will effect the game environment. In multiplayer, only operators (@) may use set commands.

Use set commands like so: /set <field> <argument(s)>.
Example command: /set desc Hi!

All set commands require a match reset to take effect.

Other command types:

  • Standard commands.
  • Option commands
  • Operator commands
